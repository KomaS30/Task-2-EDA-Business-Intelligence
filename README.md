 # Task 2 - Exploratory Data Analysis (EDA) & Business Intelligence

This project focuses on performing Exploratory Data Analysis (EDA) on the Marketing Campaign Dataset to uncover customer behavior patterns, spending trends, and business insights. Using Python and data visualization techniques, the dataset was analyzed through descriptive statistics, correlation analysis, and graphical representations such as histograms, bar charts, pie charts, heatmaps, and scatter plots.

## Objectives

* Understand the structure and quality of the dataset.
* Analyze customer demographics and purchasing behavior.
* Identify trends and patterns through visualizations.
* Perform correlation analysis to discover relationships between variables.
* Generate meaningful business insights for decision-making.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab
* GitHub

## Key Analyses Performed

* Data Loading and Exploration
* Missing Value Analysis
* Descriptive Statistics
* Univariate Analysis
* Correlation Analysis
* Customer Spending Pattern Analysis
* Data Visualization

## Visualizations Included

* Income Distribution Histogram
* Education Distribution Bar Chart
* Marital Status Pie Chart
* Correlation Heatmap
* Income vs Wine Spending Scatter Plot
* Income vs Meat Products Spending Scatter Plot

## Outcomes

The analysis provided insights into customer demographics, income distribution, purchasing habits, and relationships between different marketing attributes, helping to support data-driven business decisions.
